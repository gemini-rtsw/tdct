Database Comparison

The Perl tool DbCompare.pl allows comparison of sets of EPICS run-time databases files. DbCompare performs a " functional" comparison of the database records using information from db files.

It reports differences in
  • records
  • field values
  • record aliases
  • info nodes
New: DbCompare is now compatible with EPICS 7  (7.0.3)


Files

The tools consists of 
  • the Perl script DbCompare.pl
  • the Perl module Dbd::Parser.pm  (file Parser.pm)
Note that Parser.pm must be installed  so that it can be found by the Perl use statement
    use Dbd::Parser;
