Undo Facilities


tdct contains two undo facilities:

Undoing Drawing Actions

This section applies only to the schematic editor.

The first item on the Menu >> Edit menu allows undoing previous drawing actions (but not zooming). The menu item text shows what is being undone. This can be repeated until the the menu item text shows 'nothing to undo'.

Note that zoom operations are not part of this undo sequence.

Undoing / Redoing Zoom

This section applies both to the schematic editor and the symbol editor.

Every time a zoom rectangle is used to zoom to a part of the drawing, the viewport rectangle is remembered (up to 10 times).
Note that zoom and pan operations using the mouse-wheel and the arrow keys are not remembered.

Typing CTRL-0 allows to cycle backward through the remembered viewports.

Typing 0 reverts to the original viewport of the drawing.
