Note:
The mouse nomenclature in this section is written from the point of view of a right handed person, i.e. "left" is used to denote mouse button 1, "right" is used to denote mouse button 2.

Key actions:

zoom in  + or =
zoom out  - or _
zoom to original  0
pan canvas  arrow keys
unselect all  ESC
end of new wire draw  ESC
end of wire edit  ESC
end of property move  ESC
abort zoom by selection rectangle
(while left mouse button is still held down)		 ESC
delete selected objects  delete
undelete last  of deleted objects  shift-delete
copy selected objects to clip-board CTRL-C
cut selected objects to clip-board CTRL-X
paste selected objects from clip-board CTRL-V
undo  (enabled after cut or move of objects)
- one level of undo only		 CTRL-Z
descend hierarchy
(if one and only one symbol in a schematic is selected)		 D

Mouse wheel actions:

zoom in (with respect to centre of canvas)  CTRL scroll forward
zoom out (with respect to centre of canvas)  CTRL scroll backward
zoom in (with respect to current mouse position) CTRL-SHIFT scroll forward
zoom out (with respect to current mouse position) CTRL-SHIFT scroll backward
pan canvas vertical  scroll
pan canvas horizontal  SHIFT scroll

By starting tdct with the command line option "-z 2" this assignment is reversed.

Mouse actions when a symbol is selected:

Function Mouse Notes
descend hierarchy  left double click will open a new tab, if necessary
display property dialog right click                               only one object (symbol or comment) must
be selected
drag symbol left drag
show symbol id and symbol file name CTRL-right click only one object (symbol or comment) must
be selected
flip bus-tap SHIFT-right click flips bus-tap 180o around direction of bus-wire


Mouse actions when a wire is selected:

Function Mouse Notes
toggle edit mode  right-click if edit mode is on, selection squares are
drawn around each wire point
drag wire point left drag if you drag a wire end point and the point location allows a connection to a symbol port, the mouse cursor changes to a pointing hand.
insert wire segment at begin of wire SHIFT-left-click
insert wire segment at end of wire SHIFT-right-click
delete wire segment at begin of wire ALT-left-click
delete wire segment at end of wire ALT-right-click
show / rename wire id CTRL-right click renaming only for buses and bus-wires. Other wire ids are auto-assigned.
